# Snapchat Coin Game 🪙

A simple coin tapping game built using Snapchat Lens Studio.

## Features
- Tap the coin to earn points
- Countdown timer
- Score tracking
- High score system

## Built With
- Lens Studio
- JavaScript
- Snapchat AR

## Author
Bhavya Sri

## How to Play
1. Open the Lens.
2. Tap the coin as fast as possible.
3. Score as many points as you can before time runs out.
